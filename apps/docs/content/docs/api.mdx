# üß† CoTrain API Reference

This API allows you to query:

- üë§ Training contribution of a specific user (by wallet address)
- üß™ Real-time progress of a model training job (by model ID)
- üîê User authentication and registration
- üöÄ Training session management
- ‚õìÔ∏è Blockchain interactions
- üë• Contributor profiles

All endpoints return JSON and are accessible via `https://api.cotrain.network/v1`.

---

## üåü API Overview

### Base URLs

| Environment | Base URL | Description |
|-------------|----------|-------------|
| **Production** | `https://api.cotrain.network/v1` | Live production environment |
| **Development** | `http://localhost:3001/api` | Local development server |
| **Testnet** | `https://testnet-api.cotrain.network/v1` | Testing environment |

### Authentication

You must include your API key in the `Authorization` header for authenticated endpoints:

```http
Authorization: Bearer <your-api-key>
```

#### JWT Token Authentication
```bash
curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \
     -H "Content-Type: application/json" \
     http://localhost:3001/api/training
```

### Rate Limiting

| Tier | Requests/Minute | Burst Limit |
|------|-----------------|-------------|
| **Free** | 100 | 200 |
| **Pro** | 1,000 | 2,000 |
| **Enterprise** | 10,000 | 20,000 |

### Response Format

All API responses follow a consistent JSON structure:

```json
{
  "success": true,
  "data": {
    // Response data
  },
  "timestamp": "2024-01-15T10:30:00Z"
}
```

Error responses:
```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Validation failed",
    "details": [
      {
        "field": "email",
        "message": "Invalid email format"
      }
    ]
  },
  "timestamp": "2024-01-15T10:30:00Z"
}
```

## üìà Example Integration

### Python Example

```python
import requests
import json

# Configuration
API_BASE_URL = "https://api.cotrain.network/v1"
API_KEY = "your-api-key-here"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

# Get user contribution
def get_user_contribution(wallet_address):
    url = f"{API_BASE_URL}/users/{wallet_address}/contribution"
    response = requests.get(url, headers=headers)
    return response.json()

# Get model training progress
def get_model_progress(model_id):
    url = f"{API_BASE_URL}/models/{model_id}/progress"
    response = requests.get(url, headers=headers)
    return response.json()

# Create training session
def create_training_session(session_data):
    url = f"{API_BASE_URL}/training"
    response = requests.post(url, headers=headers, json=session_data)
    return response.json()

# Example usage
if __name__ == "__main__":
    # Check user contribution
    contribution = get_user_contribution("0xabc123...")
    print(f"User has contributed {contribution['data']['total_gpu_hours']} GPU hours")
    
    # Check model progress
    progress = get_model_progress("llama2-42b")
    print(f"Model training is {progress['data']['percent_complete']}% complete")
    
    # Create new training session
    session_data = {
        "name": "My AI Model",
        "description": "Training a custom transformer model",
        "model_type": "transformer",
        "dataset_url": "https://example.com/dataset.zip",
        "config": {
            "epochs": 5,
            "batch_size": 16,
            "learning_rate": 0.0001
        }
    }
    
    result = create_training_session(session_data)
    print(f"Created training session: {result['data']['session']['id']}")
```

### JavaScript Example

```javascript
const API_BASE_URL = 'https://api.cotrain.network/v1';
const API_KEY = 'your-api-key-here';

const headers = {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
};

// Get user contribution
async function getUserContribution(walletAddress) {
    const response = await fetch(`${API_BASE_URL}/users/${walletAddress}/contribution`, {
        headers
    });
    return await response.json();
}

// Get model training progress
async function getModelProgress(modelId) {
    const response = await fetch(`${API_BASE_URL}/models/${modelId}/progress`, {
        headers
    });
    return await response.json();
}

// Create training session
async function createTrainingSession(sessionData) {
    const response = await fetch(`${API_BASE_URL}/training`, {
        method: 'POST',
        headers,
        body: JSON.stringify(sessionData)
    });
    return await response.json();
}

// Example usage
async function main() {
    try {
        // Check user contribution
        const contribution = await getUserContribution('0xabc123...');
        console.log(`User has contributed ${contribution.data.total_gpu_hours} GPU hours`);
        
        // Check model progress
        const progress = await getModelProgress('llama2-42b');
        console.log(`Model training is ${progress.data.percent_complete}% complete`);
        
        // Create new training session
        const sessionData = {
            name: 'My AI Model',
            description: 'Training a custom transformer model',
            model_type: 'transformer',
            dataset_url: 'https://example.com/dataset.zip',
            config: {
                epochs: 5,
                batch_size: 16,
                learning_rate: 0.0001
            }
        };
        
        const result = await createTrainingSession(sessionData);
        console.log(`Created training session: ${result.data.session.id}`);
    } catch (error) {
        console.error('API Error:', error);
    }
}

main();
```
